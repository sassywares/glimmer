## Getting Started

Create a new repo using âœ¨ Glimmer as a template.

Then, it's just a Next.js app with a few extra goodies:

```bash

# Install dependencies
pnpm install

# Start the development server
pnpm dev

# Build for production
pnpm build

# Run the production build
pnpm start

# Run unit tests
pnpm test
pnpm test:watch

# Run E2E tests
pnpm e2e
pnpm e2e:watch

# Lint the codebase
pnpm lint

# Format the codebase
pnpm format

```
